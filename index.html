BucketList

BucketList is a free web application built by Lovelace Squad for General Assembly. BucketList is an application where users can log in, and set up a personal BucketList of places they want to experience.

An elegant solution designed for all users that are looking to travel and/or find things to do in the area(s) that they want to visit. We integrated with third-party API to allow users to search for a location using Foursquare’s API, and then view the search results using Mapbox to add to their BucketList.
