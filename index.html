BucketList

(Group Assignment, Fall 2015)

Web app by Lovelace Squad for General Assembly; each User creates a personal list of places.

  • Initialized application server codebase using Mongoose ODM and Passport
  • Initialized MongoDB and connected to application server
  • Integrated Bootstrap and Font Awesome icons in the client

BucketList is an elegant solution designed for all users who are looking to travel and/or find things to do in the area(s) that they want to visit. Users can log in, set up a personal BucketList of places they want to experience, and add notes.

We integrated with third-party API to allow users to search for a location using Foursquareâ€™s API, and then view the search results using Mapbox to add to the user's BucketList.
